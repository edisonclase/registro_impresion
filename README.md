# Registro Impresión

Proyecto para ajustar e imprimir libros de registro académico desde archivos Excel, respetando el formato físico de referencia en PDF.

## Reglas base
- No modificar textos oficiales del registro.
- Respetar diferencias de contenido entre grados.
- Conservar visualmente celdas vacías.
- Imprimir solo las hojas válidas hasta antes de "Reportes de Calificaciones".
- En hojas de calificaciones, imprimir número, encabezados y notas; no nombres.
- Ajustar impresión para formato físico oficial.

## Primer caso de trabajo
- Libro: 2A Segundo A 2025-2026
- Referencia: Registro físico de 2do grado en PDF
